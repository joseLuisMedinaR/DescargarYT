## Instalación
Crear un entorno virtual e instalar los requerimientos:
~~~
$ python3 -m venv .env
$ source .env/bin/activate
$ pip install -r requirements.txt
~~~
Una vez que tenemos todo instalado hay que ejecutar main.py
~~~
$ python3 main.py
~~~
